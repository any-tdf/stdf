## 2.0.0

- [!tag|A|0|] 新增 `height` 属性，支持不使用弹出层时配置展示区域高度。
- [!tag|A|0|] `popup` 支持传 `null`，可直接在页面内展示。
- [!tag|A|0|] 新增 `multiple` 多选模式，支持 `multipleSelected` 与 `onmultiplechange`。
- [!tag|A|0|] 新增 `multipleIcon` 与 `multipleIconActive`，自定义多选图标。
- [!tag|A|0|] 新增 `PickerMultipleItem` 类型，统一多选返回结构。

## 1.3.0

- [!tag|A|0|]`popup` 属性支持传入 `null`，不使用弹出层直接展示选择器。
- [!tag|A|0|]新增 `height` 属性，设置不使用弹出层时展示区域高度占页面百分比。

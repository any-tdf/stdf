# Introduction

Mobile web component library based on [Svelte](https://svelte.dev) and [Tailwind](https://www.tailwindcss.com) .

> **S**imple ‚Ä¢ **T**iny ‚Ä¢ **D**esign ‚Ä¢ **F**ast
